App.java 
// A complete TUI app in ~10 lines
public static void main(String[] args) {
  Layout root = Layout.column()
      .padding(1)
      .build();

  root.add(Text.bold("My App", 0x7F77DD));
  root.addFlex(
    Table.builder(model, columns).build()
  );
  root.add(statusBar);

  TerminusApp.run(root);
}
features

Everything a TUI needs.
Nothing it doesn't.

01 · components

Composite model

Everything is a Component. Compose Leaf nodes into Container trees. Uniform tree traversal and focus propagation.

02 · layout

Declarative flex layout

Flex row/column with gap, padding, and alignment. Fixed or flex-sized columns. No manual coordinate math.

03 · performance

60fps double-buffered

Double-buffered diff rendering. Only changed cells hit the terminal. Zero flicker, even on slow connections.

04 · table

Virtual scrolling

Tables with 100k+ rows render in O(visible rows). Numeric and lexical sort, multi-cycle column cycling, selection callbacks.

05 · input

Rich keyboard & mouse

Full ANSI escape sequence parser. SGR mouse support. TextInput with history, kill rings, word-jump, and selection.

06 · runtime

Zero runtime deps

Only JNA for raw terminal mode. No logging framework, no JSON library, no HTTP client. Drop it in and ship.

install

One dependency.
Full framework.

Add terminus-core to your build. No annotation processors, no code generation. Just components.

GitHub →
Gradle
// build.gradle
dependencies {
  implementation 'io.terminus:terminus-core:1.0.0'
}
Maven
<dependency>
  <groupId>io.terminus</groupId>
  <artifactId>terminus-core</artifactId>
  <version>1.0.0</version>
</dependency>
Run the demo
$ git clone https://github.com/P0intMaN/terminus.git
$ cd terminus
$ ./gradlew :terminus-demo:shadowJar
$ ./run-demo.sh   # real terminal — not via gradle
